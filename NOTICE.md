# 자료 저작권 및 출처 고지

본 저장소의 `data/raw/`, `data/converted/`, `data/ocr/` 하위 자료는
각 대학 입학처(또는 대학)에서 발행·공개한 자료이며, 저작권은 각 대학에
속한다. 본 저장소는 고등학교 진학지도라는 비영리 교육 목적의 인덱스를
제공할 뿐, 자료의 저작권을 주장하지 않는다.

## 자료별 허가 상태

자료별 허가 상태는 `data/meta.yaml`의 `license_status` 필드로 관리한다.

- `confirmed` — 해당 대학 입학처로부터 명시적인 재배포·인용 허가를 받음
- `pending`   — 회신 대기 중. 빌드 시 사이트에는 “원본 링크만” 모드로 노출
- `link_only` — 허가받지 못함 또는 미요청. 사이트에는 공식 페이지 링크만 노출하며
                 원본 파일은 호스팅하지 않음 (GitHub Pages 빌드 시 제외)
- `denied`    — 거부됨. 저장소에서 제거

## 삭제 요청

저작권자(대학 입학처)께서 자료의 삭제를 요청하시는 경우,
GitHub Issue 또는 저장소 관리자 이메일로 연락 주시면 즉시 처리한다.

## 출처 표기 원칙

- 사이트의 모든 자료 카드에는 “원본: <대학명> 입학처” 표기를 의무로 한다.
- 가능한 경우 각 대학 입학처 페이지로의 직접 링크(`official_url`)를 함께 제공한다.
- 자동 추출 수치는 “추정값” 표지와 함께 표시하며, 권위 있는 값은 항상 원본임을 명시한다.
